Tabula Rasa: a blank slate for WordPress development

Tabula Rasa is a blank theme/starter theme/theme framework for custom WordPress development. It aims to help you develop better themes faster.

Get Started

  1. Download the theme archive above
  2. Extract to your themes directory
  3. Edit the stylesheet header with your project's details. That's it.
